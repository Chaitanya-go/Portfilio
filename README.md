# Portfolio

Modern, responsive, and interactive personal portfolio website for Chaitanya Joshi.

## Features
- **Interactive Particle Background**: Canvas-based particles with mouse interaction.
- **Custom Cursor**: Stylish neon dot and outline.
- **Dark Mode Aesthetic**: Sleek glassmorphism and neon accents.
- **Responsive Design**: Optimized for all devices.
- **Typing Effect**: Dynamic display of professional roles.
- **Contact Form**: Integrated with Web3Forms for backend-less email handling.

## Deployment
This is a static site. You can deploy it using GitHub Pages, Vercel, or Netlify by simply uploading the files.
